# EasyNotes

EasyNotes — це консольний застосунок на мові програмування C++ для керування
текстовими нотатками.

## Опис проєкту
Програма дозволяє користувачу додавати, переглядати та видаляти нотатки.
Збереження даних здійснюється у файл.

## Технології
- Мова програмування: C++
- Стандарт: C++17
- Тип застосунку: консольний

## Запуск проєкту
1. Відкрити проєкт у середовищі розробки (Visual Studio, CLion).
2. Скомпілювати файл `main.cpp`.
3. Запустити програму з консолі.
